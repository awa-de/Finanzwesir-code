# Build-Ziel: statische Homepage

Hier soll später die lokal testbare statische HTML-Seite entstehen.

Ziel dieser Werkstattphase:

```text
Eine vollständige HTML-Datei, die lokal im Browser geöffnet werden kann
und so wirkt, wie die Homepage später für Nutzer wirken soll.
```

Nicht Teil dieses Ordners:

- Ghost-Template-Zerlegung
- Handlebars-Partials
- Routing
- Deployment

Empfohlener späterer Dateiname:

```text
finanzwesir-homepage-funnel-v1.html
```
