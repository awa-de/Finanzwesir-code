## Die klare Empfehlung
Option 2 – nach Arbeitspaket ist für deinen Workflow optimal, und zwar aus einem Grund, der in der Tabelle nicht vollständig sichtbar wird:
Ein Arbeitspaket ist semantisch abgeschlossen. Claude versteht was gebaut werden sollte, kann den Code dagegen prüfen, und das Review hat einen klaren Scope.
Ein Review über mehrere Pakete ist wie ein Buchhalter, der erst nach drei Monaten die Belege prüft – möglich, aber schmerzhaft.
