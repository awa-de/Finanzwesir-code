## 📝 Formatierung-Standards

### Überschriften

**H1 (nur 1x pro Seite):**
```markdown
# Aktienquote: Wie viel Risiko verträgt Ihr Portfolio?
```

**H2 (Hauptabschnitte):**
```markdown
## Was ist die Aktienquote?
## Wie finde ich meine Aktienquote?
## Häufige Fehler bei der Aktienquote
```

**H3 (Unterabschnitte):**
```markdown
### Konservativ (30-50% Aktien)
### Moderat (50-70% Aktien)
### Offensiv (70-100% Aktien)
```

**Keine H4, H5, H6** – Wenn nötig, Umstrukturierung.

---

### Listen

**Ungeordnete Listen:**
```markdown
- Punkt 1
- Punkt 2
- Punkt 3
```

**Geordnete Listen:**
```markdown
1. Erster Schritt
2. Zweiter Schritt
3. Dritter Schritt
```

**Keine verschachtelten Listen** – Flach halten.

---

### Zitate & Hervorhebungen

**Blockquote (für Zitate):**
```markdown
> "The stock market is a device for transferring money from the impatient to the patient."  
> — Warren Buffett
```

**Fettdruck (für Betonung):**
```markdown
Das **Weltportfolio** besteht aus nur 2 ETFs.
```

**Kursiv (für Begriffe):**
```markdown
Der *Total Expense Ratio* (TER) liegt bei 0,2%.
```

---

### Zahlen & Daten

**Prozent:**
```markdown
Die Aktienquote liegt bei 70%.
```

**Geldbeträge:**
```markdown
50.000€ Vermögen
```

**Zeiträume:**
```markdown
2008–2024 (Bindestrich, nicht Gedankenstrich)
```

**Keine Tausenderpunkte bei 4-stelligen Zahlen:**
```markdown
1500€ (nicht 1.500€)
50.000€ (ja, Punkt ab 5 Stellen)
```

---

### Links

**Interne Links (absolut):**
```markdown
Mehr dazu: [Weltportfolio](/weltportfolio)
```

**Externe Links (mit Attribut):**
```markdown
Quelle: [MSCI World Factsheet](https://www.msci.com/world)
```

**Download-Links:**
```markdown
📥 [Kochrezept PDF herunterladen](/kochrezept.pdf)
```

## 🖼️ Infografik-Integration

### Platzierung

**Nach H2-Überschrift:**
```markdown
## Wie finde ich meine Aktienquote?

[INFOGRAFIK: Risiko-Rechner]

Die Aktienquote hängt von...
```

**Nicht mitten im Absatz** – immer vor oder nach vollständigem Textblock.

---

### Beschriftung

**Jede Infografik braucht:**
1. **Titel** (H3 oder Bildunterschrift)
2. **Kurzbeschreibung** (1-2 Sätze)
3. **CTA** (optional, z.B. "Probieren Sie den Rechner aus")

**Beispiel:**
```markdown
### Interaktiver Risiko-Rechner

Finden Sie heraus, wie viel Verlust Sie verkraften können.  
Geben Sie Ihr Vermögen ein und sehen Sie sofort das Ergebnis.

[INFOGRAFIK: Risiko-Rechner]

**Hinweis:** Dieser Rechner dient nur zur Orientierung und ersetzt keine individuelle Beratung.
```

## ✅ Content-Checkliste

**Vor Veröffentlichung jeder Seite:**

### Struktur
- [ ] H1-Überschrift vorhanden (nur 1x)
- [ ] H2-Abschnitte logisch strukturiert
- [ ] Keine H4/H5/H6 (nur H1-H3)
- [ ] Absätze nicht länger als 5 Zeilen
- [ ] Listen wo möglich (statt lange Fließtexte)

### Stil
- [ ] Sie-Form durchgängig
- [ ] Mary-Poppins-Tonalität (siehe PROJECT_CORE.md)
- [ ] Keine Übertreibungen ("unglaublich", "revolutionär")
- [ ] Keine Versprechungen ("garantiert", "sicher")
- [ ] Finanzwesir-Stil: sachlich, aber nicht trocken

### Links
- [ ] Mindestens 3 interne Links pro Seite
- [ ] Vorherige/Nächste Seite verlinkt (wenn vorhanden)
- [ ] Kochrezept + Tools verlinkt (am Ende)
- [ ] Externe Links mit Quelle versehen

### Infografiken
- [ ] Mindestens 1 Infografik pro Hauptseite (Seiten 1-9)
- [ ] Infografik-Titel vorhanden
- [ ] Kurzbeschreibung vorhanden
- [ ] Nach H2-Überschrift platziert

### SEO
- [ ] Title (60 Zeichen)
- [ ] Meta Description (155 Zeichen)
- [ ] URL-Slug sprechend (`/aktienquote`, nicht `/seite-4`)
- [ ] Alt-Text für Infografiken

### Rechtliches
- [ ] Disclaimer vorhanden (siehe PROJECT_CORE.md)
- [ ] "Keine Anlageberatung"-Hinweis
- [ ] Quellen zitiert (bei Daten/Statistiken)

## 🔍 SEO-Standards

### Title-Tag

**Format:** `[Thema]: [Benefit] – Finanzwesir`

**Beispiele:**
```
Weltportfolio: So investieren Sie global diversifiziert – Finanzwesir
Aktienquote: Wie viel Risiko verträgt Ihr Portfolio? – Finanzwesir
Kaufen: Der richtige Broker für ETF-Sparpläne – Finanzwesir
```

**Länge:** Max. 60 Zeichen (inkl. " – Finanzwesir")

---

### Meta Description

**Format:** [Problem] + [Lösung] + [CTA]

**Beispiele:**
```
Sie wissen nicht, wie viel Aktien in Ihr Portfolio gehören? Finden Sie mit unserem Risiko-Rechner Ihre optimale Aktienquote. Jetzt starten!

ETF-Sparplan starten, aber welcher Broker? Vergleichen Sie Kosten, Sparplan-Angebot und Benutzerfreundlichkeit. Jetzt informieren!
```

**Länge:** Max. 155 Zeichen

---

### URL-Slugs

**Regeln:**
- Nur Kleinbuchstaben
- Bindestriche statt Leerzeichen
- Keine Umlaute (ä → ae, ö → oe, ü → ue, ß → ss)
- Kurz und prägnant (1-3 Wörter)

**Beispiele:**
```
/der-weg
/weltportfolio
/aktienquote
/haeufige-fehler
/etf-aera
```

**Nicht:**
```
/die-aktienquote-im-portfolio
/haeufige_fehler_beim_investieren
/ETF-Aera
```
